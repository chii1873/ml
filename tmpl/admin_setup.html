(10)メールヘッダ指定
  • 配信されるメールの冒頭(ヘッダ)に付加する情報を指定します。
  • (5)のパラメータ設定のところで「付加しない」に設定されている場合は、以下の文字列の指定にかかわらず付加されません。
  • 以下の文字列を使うと、それぞれシステムに設定されている値を埋め込みます。
    • ##title## … このシステムの名称
    • ##k_email## … 管理人のメールアドレス
    • ##ml_dir## … このメーリングリストを置くディレクトリ
    • ##ml_cnt## … 投稿番号
    • ##home## … あなたのメインホームページのURL
(11)メールフッタ指定
  • 配信されるメールの末尾(フッタ)に付加する情報を指定します。
  • (5)のパラメータ設定のところで「付加しない」に設定されている場合は、以下の文字列の指定にかかわらず付加されません。
  • 以下の文字列を使うと、それぞれシステムに設定されている値を埋め込みます。
    • ##title## … このシステムの名称
    • ##k_email## … 管理人のメールアドレス
    • ##ml_dir## … このメーリングリストを置くディレクトリ
    • ##ml_cnt## … 投稿番号
    • ##home## … あなたのメインホームページのURL
    • ##idpw## … 簡易暗号化された購読者のIDとパスワード(個々のメールに差し込まれます)
